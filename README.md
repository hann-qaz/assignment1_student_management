## Project Overview
This project is a simple Java program.
It shows basic Object-Oriented Programming concepts.
The program uses classes, objects, arrays, and methods.

## Classes

### Student
The Student class stores student information.
It has name, id, major, GPA, and credits.
All fields are private and accessed using methods.

### Course
The Course class represents a course.
It contains an array of Student objects.
This shows composition in Java.
